VR失智老人照護訓練應用程式

本專案為與 高雄榮總合作 的 VR 訓練系統，針對失智症患者照護、環境安全辨識與醫療流程教學而設計。
系統使用 Unity 與 XR Interaction Toolkit 開發，建立可互動的 VR 模擬環境，協助醫護人員與相關實習生在低風險情境下進行訓練。

📌 系統目標

提供安全的訓練場域：讓醫護人員可在無風險的虛擬場景中進行操作與評估

模擬失智症照護場景：包含迷走、跌倒風險、居家環境危害辨識

教育與流程複習：透過步驟式引導協助學員學習標準操作流程

行為蒐集與紀錄：可擴充紀錄使用者的互動行為與反應時間，用於後續分析

🧠 系統功能

● 1. VR 環境建置

建立失智友善場景、居家氛圍、花園等等

● 2. XR Interaction 互動

透過 XR Interaction Toolkit 實作：

手把 Raycast 點擊

物件檢查（如拾取、旋轉、檢視）

UI 操作（確認、跳過、提示面板）

● 3. 教學與流程引導

使用者依照提示完成訓練流程，例如：

按照步驟完成照護處置

收到語音或文字提示進行下一步行動


🛠 使用技術

Unity2022.3.52f1 

C#

XR Interaction Toolkit

Oculus / OpenXR

導覽式 UI、互動流程控制、事件系統

自訂互動判斷與回饋系統

▶️ 執行方式

Clone 或下載專案

使用 Unity Hub 開啟

確認啟用 XR Plugin Management（OpenXR 或 Oculus）

連接 VR 裝置

執行場景 TrainingScene.unity

👤 計畫背景

本專案與 高雄榮總（Kaohsiung Veterans General Hospital） 合作，
作為醫護教育輔助工具，用於：

失智症照護教育

模擬情境教學（無風險）

透過 VR 提供沉浸式體驗，強化學習效果與臨場判斷能力。
